# Task Tracker (js-arrays-part1-variant3)

Система управління завданнями, написана на JavaScript. Дозволяє створювати завдання, фільтрувати їх, сортувати та відстежувати статистику виконання.

## Функціонал
* **Створення завдань**: Зберігання назви, опису, пріоритету, статусу та дедлайну.
* **Фільтрація**: Пошук за статусом, пріоритетом або категорією.
* **Сортування**: За терміном виконання або рівнем важливості.
* **Пошук**: Повнотекстовий пошук за заголовком або описом.
* **Аналітика**: Автоматичний підрахунок прострочених завдань та відсотка виконання.

## Технічні особливості
* Використання методів масивів: `filter`, `sort`, `map`, `reduce`.
* Статуси: `todo`, `in-progress`, `done`.
* Автоматична перевірка дати (deadline) відносно поточного часу.
